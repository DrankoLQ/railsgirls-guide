3. Creando las fotos de la aplicación

En esta edición de Rails Girls Córdoba 2015 crearás la aplicación web pic-home, en la que los usuarios podrán subir fotografías y comentar, “megustear” las fotografías de otros usuarios.

  • Crea el modelo photo a través de la utilidad scaffold de rails:

    rails generate scaffold photo name:string description:string picture:string

    • Con la utilidad scaffold estás creando en un solo paso la estructura de datos de tus photos
    • Abre un navegador de internet y escribe la dirección web http://localhost:3000 para ver la aplicación. Como puedes ver, por ahora tu aplicación solo tiene una página de relleno.
    • Siempre que quieras cerrar el servidor de rails, pulsa las teclas CTRL y C en la terminal.
  • Crea las tablas de la base de datos con el comando rake:

    rake db:migrate

    • Con este comando estás creando tablas donde almacenar la información de cada photo que subas a tu aplicación web.
  • Lanza el servidor de rails para ver tu aplicación con los cambios que has realizado:

    rails server

    • Si ya tenías el servidor de rails lanzado, cierralo. Para cerrar el servidor de rails pulsa las teclas CTRL y C en la terminal.
    • Dedica un rato a explorar las siguientes direcciones web de tu aplicación:
      • http://localhost:3000/photos
      • http://localhost:3000/photos/new
