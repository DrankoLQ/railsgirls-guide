5. Haciendo una aplicación web bonita

El aspecto que tiene ahora tu aplicación web no es precisamente el mejor del mundo, ¿verdad? Hasta ahora te has ocupado de una parte esencial de tu aplicación: que funcione. Ahora toca ponerla bonita para que sea atractiva de usar. Para ello tendrás que modificar la estructura de los ficheros HTML y utilizar hojas de estilo CSS.

  • Copia el contenido de pic-home-beauty.zip en sus carpetas correspondientes.
    • Revisa el contenido de cada fichero que has reemplazado para ver las cosas que han cambiado. No dudes en llamar a un mentor para que te explique todo.
  • Abre el fichero app/controllers/photos_controller.rb y busca la siguiente línea de código...

    def index

    … Y añade justo debajo la siguiente línea de código:

    @photo = Photo.new

    • Esta línea de código crea un contenedor para una foto nueva. Con la nueva estructura de HTML, las fotos se pueden crear desde la lista de fotos, por lo que es necesario incluir este contenedor, para alojar la foto nueva y que rails se encargue de guardarla.
