1. Creando la aplicación web

En esta edición de Rails Girls Córdoba 2015 crearás la aplicación web pic-home, en la que los usuarios podrán subir fotografías y comentar, "megustear” las fotografías de otros usuarios.

Para empezar, crea una aplicación básica siguiendo los siguientes pasos:

  • Abre una terminal para lanzar comandos.
    • Windows: En Menú Inicio, busca y ejecuta Command Prompt with Ruby and Rails.
    • Mac OS X: Abrir Spotlight, busca y ejecuta Terminal.
    • Linux: Busca y ejecuta Terminal.
  • Crea la nueva aplicación de rails con el siguiente comando:

    rails new pic-home

    • Este comando crea un nuevo proyecto de rails en una carpeta llamada pic-home.
    • Este paso puede tardar un poco debido a que rails está descargando todas las dependencias necesarias para funcionar.
  • Entra en el proyecto de la aplicación que acabas de crear:

    cd pic-home

  • Lanza el servidor de rails para ver tu aplicación recién creada:

    rails server

    • Con este comando, el servidor de rails se prepara para atender a los visitantes de tu aplicación web. ¡Si el servidor no está lanzado, nadie podrá ver tu aplicación!
    • Abre un navegador de internet y escribe la dirección web http://localhost:3000 para ver la aplicación. Como puedes ver, por ahora tu aplicación solo tiene una página de relleno.
    • Siempre que quieras cerrar el servidor de rails, pulsa las teclas CTRL y C en la terminal.
